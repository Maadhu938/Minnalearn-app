# 🇯🇵 MinnaLearn – Flutter Japanese Learning App

**MinnaLearn** is a Flutter-based mobile application designed for **JLPT N5 learners**.  
It helps users learn **Japanese vocabulary, kana, and basic kanji** through **flashcards, quizzes, and interactive mini-games** inspired by the *Minna no Nihongo* learning approach.

The goal of the app is to make Japanese learning **engaging, interactive, and beginner-friendly**.

---

## ✨ Features

### 📚 Lessons
- Structured lessons based on JLPT N5 level
- Vocabulary and basic kanji for each chapter
- Beginner-friendly explanations

### 🃏 Flashcards
- Interactive vocabulary practice
- Shows reading and meaning
- Designed for quick revision

### 🎮 Learning Games
Gamified learning to reinforce memory:
- **Matching Game** – Match Japanese words with meanings  
- **Memory Game** – Test recall ability  
- **Typing Game** – Practice correct spelling  
- **Word Catch Game** – Fast-paced recognition game

### 📝 Quizzes
- Lesson-based quizzes
- Reinforces learning after completing lessons

### 📊 Progress Tracking
- Tracks completed lessons
- Records quiz and game scores
- Helps monitor learning progress

### 🎨 Modern UI
- Clean and responsive Flutter UI
- Smooth animations and user-friendly design

## 🚀 Getting Started

Clone the repository:
```sh
git clone https://github.com/Maadhu938/MinnaLearn-FlutterApp.git
```

Navigate to the project directory:
```sh
cd MinnaLearn-FlutterApp/flutter_minnalearn
```

Install dependencies:
```sh
flutter pub get
```

Run the application:
```sh
flutter run
```
